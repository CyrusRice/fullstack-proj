Description

This is a full stack chess web app developed by Vinodh Kotipalli and Cyrus Rice in winter 2021. You can create an account, add friends, and play with those friends, among other features. This app is deployed using Heroku.

Implementation

  • Front-End

    The front end was implemented in html, css, and javascript. Below are some javascript libraries that were used.

    • chess.js
      • Used to implement chess game logic, such as move validation and board status
      • Link
    • chessboard.js
      • Used to display board in html
      • Link
    • socketio.js
      • Used to implement web sockets for the client.
      • Link

  • Back-End

    The backend was implemented in Python. Below are some libraries that were used.

    • flask
      • Used as backend server.
      • Link
    • flask-socketio
      • Used for communicating with front-end sockets
      • Link
    • mongodb
      • Used to store data such as user and game data
      • Link
    • pymongo
      • Used to communicate with the database from the backend using python
      • Link

Basic Rules

Pieces

Piece Legal Move Legal Attack Special
Pawn Up 1 square Upper-Left or Upper-Right 1 square On pawns first move, can move 2 squares forward
King 1 square in all directions Same as move Can't move into/must move out of a checkmate position Castling - if rook and king in original positions, the king is moved two squares toward the rook, and the rook moves past the king to the square right next to where the king has moved.
Queen Unlimited squares in any direction Unlimited squares in any direction n/a
Knight up/down 1 left 2 up/down 1 right 2 right/left 1 up 2 right/left 1 down 2 Same as move n/a
Bishop Diagonal in any direction unlimited squares Same as move n/a
Rook Straight in any direction unlimited squares Same as move Castling - if rook and king in original positions, the king is moved two squares toward the rook, and the rook moves past the king to the square right next to where the king has moved.
